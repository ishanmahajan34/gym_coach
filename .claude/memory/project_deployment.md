Render is the active deployment platform (replaced Fly.io).

**Config file:** `render.yaml` (Render Blueprint) — declares the web service + 1 GB persistent disk.

**Disk:** Mounted at `/data`, `DB_URL=sqlite+aiosqlite:////data/coach.db`. Requires at least the Starter plan ($7/mo) — free tier has no persistent disk.

**PORT:** Render injects `$PORT`; Dockerfile CMD uses `${PORT:-8000}`.

**Secrets to set manually in Render dashboard** (not in render.yaml):
- `SECRET_KEY` — auto-generated by `generateValue: true` in render.yaml
- `TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`, `GEMINI_API_KEY` — set when M4/M6 is ready

**PUBLIC_BASE_URL** in render.yaml defaults to `https://gym-coach.onrender.com` — update to the actual subdomain after first deploy.

**Deploy:** Push to the connected Git repo branch (main); Render auto-builds via Docker.

**Why:** Moved from Fly.io to Render (user opened a Render account 2026-06-04).
**How to apply:** Any deploy or hosting questions should reference Render, not Fly.io.
